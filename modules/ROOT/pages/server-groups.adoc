== About This Document

This document, as well as the rest of the documents that make up the xref:3.4@rest-api-reference.adoc[Mule Management Console REST API Reference Guide], are a technical reference only. This document does not provide contextual information such as instructions, use cases, or scenarios. To understand this document, you should be familiar with Mule Runtime, Server Groups, and the xref:3.4@using-the-management-console-api.adoc[REST API]

/api/serverGroups
----

[%autowidth.spread]
|===
|*POST* | |*CREATE*
| | |
|===

Creates a new Server Group.

[.ex]
=====
[discrete.view]
=== -> Request

[discrete]
==== JSON

[source,text,linenums]
----

{
  "name": "NewGroupName"
}
----

[%header%autowidth.spread]
|===
|Key |Type |Summary |Child of
|name |string |Name of the Server Group to be created |--
|===

[discrete]
===== Using CURL on Windows

[source,text,linenums]
----
curl --basic -u admin:admin -d "{ \"name\" : \"NewGroupName\" }" --header "Content-Type: application/json" http://localhost:8080/mmc/api/serverGroups
----

[discrete]
===== Using CURL on Linux

[source,text,linenums]
----
curl --basic -u admin:admin -d '{ "name" : "NewGroupName" }" --header 'Content-Type: application/json' http://localhost:8080/mmc/api/serverGroups
----

===  <- Response

[discrete]

[source,json,linenums]
----
{
  "name": "NewGroupName",
  "id": "73d89173-290e-4cb3-a61c-e11deb74767d",

|===
|Key |Type |Summary |Child of

|href |String |Full link to the Server Group resource on which you can perform an operation |--
|===
