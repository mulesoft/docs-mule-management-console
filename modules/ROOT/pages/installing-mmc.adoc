*xref:installing-the-production-version-of-mmc.adoc[Install the production version] if you are planning to use the Mule Management Console in production.*

* You need a valid enterprise license to launch the production version of MMC.
* You need to have the current version of Mule Enterprise already installed.
* The production version of MMC is packaged as a .war file designed to run in an application server or container, such as xref:installing-the-production-version-of-mmc.adoc[Tcat Server], xref:installing-the-production-version-of-mmc.adoc[Apache Tomcat], or xref:installing-the-production-version-of-mmc.adoc[JBoss].

== See Also

Once you have installed the Mule Management Console:

. Get familiar with the xref:orientation-to-the-console.adoc[MMC console].
. Learn the basics of using MMC with the xref:mmc-walkthrough.adoc[MMC Walkthrough].
. Learn more about how to xref:setting-up-mmc.adoc[set up MMC] to meet your needs.

Already have MMC installed, but need to update it to a new version? Find out xref:upgrading-the-management-console.adoc[how to upgrade].
